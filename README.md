# ML-stepic-classic

Реализация классических алгоритмов машинного обучения **с нуля**  
в рамках курса [«Алгоритмы Машинного обучения с нуля»](https://stepik.org/course/68260/syllabus) на Stepik.

Цель проекта — глубоко понять принципы работы базовых ML-алгоритмов, реализовав их **без использования высокоуровневых библиотек**, таких как Scikit-learn, TensorFlow или PyTorch.  
Каждый алгоритм оформлен как отдельный Python-класс с интерфейсом, похожим на Scikit-learn (`fit`, `predict`, `score` и т.д.).

---

## Содержание

В репозитории вы найдёте реализации следующих алгоритмов:
- Линейные модели (**LinearRegression**, **LogisticRegression**, **SVM**)
- Метрические алгоритмы (**KNNClassifier**, **KNNReg**)
- Деревья решений (**TreeReg**, **TreeClf**)
- Ансамбли (Бэггинг, Случайный лес, Градиентный бустинг)
- Снижение размерности (**PCA**)

> Все реализации используют только **NumPy** и **Pandas**, что делает код прозрачным и образовательным.
